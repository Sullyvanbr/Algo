Exercice 1 : Recherche séquentielle simple Écrire un algorithme qui : 
- Crée un tableau de prénoms - Demande un prénom à rechercher 
- Utilise la recherche séquentielle pour le trouver 
- Affiche sa position ou "non trouvé"

FONCTION  rechercheSequentielle(tableau, element)
    POUR i de 0 à longueur(tableau) -1
	SI tableau[i] = element
	    RETOURNER i 
	FIN SI
    FIN POUR
    RETOURNER -1 
    AFFICHER pour -1 'Non trouvé'
FIN FONCTION

tableau <- [Prenoms]
element <- prénom dans le tableau demandé
