Exercice 2 : Notes de plusieurs classes
Créer un tableau représentant les notes de 3 classes avec 5 élèves chacune.
Calculer la moyenne d'une classe spécifique
Calculer la moyenne générale de toutes les classes


FONCTION sommerLigne(tableau, Classes)
    somme <- 0
    POUR j de 0 à nombreDeColonnes(tableau[Classes]) -1
	somme <- somme + tableau[Classes][j] / nombre de notes d'une ligne
    FIN POUR
    RETOURNER somme
FIN FONCTION
